# Treasure Game 🗺️

Welcome to **Treasure Island**!  
This is a text-based adventure game built in Python as part of Angela Yu’s **100 Days of Code** challenge.  
The player makes choices to navigate through an island in search of hidden treasure — but beware, one wrong move can end the journey!

---

## 📖 Description
Treasure Game is a beginner-friendly Python project that demonstrates the use of:
- Conditional statements (`if/elif/else`)
- User input handling
- Branching storylines with multiple outcomes

It’s a fun way to practice programming fundamentals while building an interactive console game.

---

## 🎮 Features
- Interactive console-based gameplay
- Multiple story paths and endings
- Beginner-friendly Python code
- Great starter project for learning GitHub and Python basics
