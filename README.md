# Treasure Game 🗺️

A text-based adventure game built in Python as part of the **100 Days of Code** challenge.  
The player makes choices to find hidden treasure — but one wrong move can end the journey!

## Features
- Interactive console-based gameplay
- Multiple story paths and endings
- Beginner-friendly Python code using conditionals and input
- Great starter project for learning GitHub and Python basics
